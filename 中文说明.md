Dakava是用Node.js开发的web应用开发工具。Dakava用于设计数据模型并能生成RESTful风格的API，此外Dakava还有一个可以使用桌面浏览器访问的简洁Web界面。
* 演示地址 : http://node.jprda.com/greeting/myTable

#### 使用技术
* 数据库 : MongoDB
* IDE : Codebox
* 开发语言 : Node.js
* 开发框架 : Sails
* Web UI : Kendo UI
* 版本控制 : Git
